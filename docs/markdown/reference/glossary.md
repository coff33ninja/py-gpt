# Glossary

Common terms and concepts used in PyGPT and AI.

## A

**Agent**
An AI system that can autonomously execute multi-step tasks, use tools, and make decisions to achieve goals.

**API (Application Programming Interface)**
A set of protocols that allows different software applications to communicate. PyGPT uses APIs to connect to AI providers.

**API Key**
A unique identifier used to authenticate requests to an API service. Required to use most AI models.

**Anthropic**
AI company that created Claude models, focusing on AI safety and reliability.

**Attachment**
A file (document, image, code, etc.) that can be included with a message for AI analysis.

**Autonomous Mode**
A mode where AI operates independently with minimal human intervention to achieve a goal.

## B

**Batch Processing**
Processing multiple items or requests together, often more efficient than one-by-one.

**Beam Search**
A search algorithm used in text generation to find the most likely sequences.

**BERT**
Bidirectional Encoder Representations from Transformers, a type of language model.

## C

**Chat Mode**
The primary mode for conversational interaction with AI models.

**ChatGPT**
OpenAI's conversational AI interface, similar to PyGPT's chat functionality.

**Chunk**
A segment of text created when splitting documents for embedding or processing.

**Claude**
Anthropic's family of AI models known for safety and reasoning capabilities.

**Code Interpreter**
A feature that allows AI to write and execute Python code in real-time.

**Completion**
A mode where AI continues or completes text based on a prompt.

**Context**
The conversation history and information available to the AI model.

**Context Window**
The maximum amount of text (in tokens) a model can process at once.

**Custom Command**
A user-defined command with a specific prompt template for quick execution.

## D

**DALL-E**
OpenAI's image generation model.

**DeepSeek**
AI provider offering cost-effective language models.

**Deterministic**
Producing the same output given the same input (temperature = 0).

**Docker**
A platform for running applications in isolated containers, used for safe code execution.

## E

**Embedding**
A numerical representation of text that captures semantic meaning, used in vector search.

**Endpoint**
The URL where API requests are sent.

**Expert**
A specialized AI agent with a specific role in multi-agent collaboration.

## F

**Few-Shot Learning**
Providing a few examples in the prompt to guide the model's behavior.

**Fine-Tuning**
Training a model on specific data to specialize its behavior.

**Frequency Penalty**
A parameter that reduces repetition in generated text.

**Function Calling**
The ability of AI models to call predefined functions or tools.

## G

**Gemini**
Google's family of multimodal AI models.

**GPT (Generative Pre-trained Transformer)**
A type of large language model developed by OpenAI.

**Grok**
xAI's AI model with real-time information access.

**Grounding**
Connecting AI responses to real-world data or sources.

## H

**Hallucination**
When an AI generates false or nonsensical information presented as fact.

**HuggingFace**
A platform hosting thousands of AI models and datasets.

**Hybrid Search**
Combining vector search with keyword search for better results.

## I

**Imagen**
Google's image generation model.

**Index**
A database of embedded documents for semantic search.

**Inference**
The process of using a trained model to generate outputs.

**IPython**
An interactive Python shell used in Code Interpreter.

## J

**JSON (JavaScript Object Notation)**
A data format used for structured information exchange.

**JSON Mode**
A model setting that ensures output is valid JSON.

## K

**Kernel**
The computational engine that executes code in Code Interpreter.

**k-NN (k-Nearest Neighbors)**
An algorithm used in vector search to find similar items.

## L

**Latency**
The delay between sending a request and receiving a response.

**LLM (Large Language Model)**
An AI model trained on vast amounts of text data to understand and generate language.

**LlamaIndex**
A framework for connecting LLMs with external data sources.

**Llama**
Meta's family of open-source language models.

**Loop Mode**
Continuous conversation mode without manual microphone activation.

## M

**Max Tokens**
The maximum number of tokens in a model's response.

**MCP (Model Context Protocol)**
A protocol for connecting to remote tools and services.

**Mistral**
A European AI company offering fast, capable models.

**Multimodal**
Capable of processing multiple types of input (text, images, audio, video).

## N

**Neural Network**
A computing system inspired by biological neural networks, basis of modern AI.

**Nucleus Sampling (Top P)**
A text generation method that samples from the most likely tokens.

## O

**OCR (Optical Character Recognition)**
Technology for extracting text from images.

**Ollama**
A platform for running AI models locally on your computer.

**OpenAI**
AI research company that created GPT models and DALL-E.

**OpenRouter**
A service that provides unified access to multiple AI providers.

## P

**Parameter**
A value in a neural network learned during training, or a setting that controls model behavior.

**Perplexity**
An AI provider specializing in research and web search.

**Plugin**
An extension that adds new features or capabilities to PyGPT.

**Presence Penalty**
A parameter that encourages the model to explore new topics.

**Preset**
A saved configuration including system prompt, model, and settings.

**Prompt**
The input text given to an AI model to generate a response.

**Prompt Engineering**
The practice of crafting effective prompts to get desired outputs.

## Q

**Qdrant**
A vector database for storing and searching embeddings.

**Query**
A search request or question sent to a system.

**Queue**
A line of requests waiting to be processed.

## R

**RAG (Retrieval-Augmented Generation)**
A technique that retrieves relevant information before generating a response.

**Rate Limit**
The maximum number of requests allowed in a time period.

**ReAct (Reasoning + Acting)**
An agent framework that combines reasoning with action execution.

**Realtime API**
An API for low-latency, streaming interactions.

**Remix**
Modifying or extending a previously generated image or video.

**Response API**
OpenAI's API for structured AI interactions.

## S

**Sampling**
The process of selecting tokens during text generation.

**Semantic Search**
Search based on meaning rather than exact keyword matching.

**Sonar**
Perplexity's family of research-focused models.

**Streaming**
Receiving a response incrementally as it's generated.

**System Prompt**
Instructions that define the AI's role and behavior.

## T

**Temperature**
A parameter controlling randomness in text generation (0 = deterministic, 2 = very random).

**Token**
A unit of text (roughly 4 characters or 0.75 words) used by language models.

**Tool**
A function or capability that an AI agent can use to perform actions.

**Top K**
The number of most likely next tokens considered during generation.

**Top P (Nucleus Sampling)**
A sampling method that considers tokens until their cumulative probability reaches P.

**Transformer**
A neural network architecture that powers modern language models.

**TTS (Text-to-Speech)**
Technology that converts text into spoken audio.

**Truncation**
Cutting off text when it exceeds a length limit.

## U

**Upscale**
Increasing the resolution or quality of an image.

## V

**Vector**
A numerical representation of data in multi-dimensional space.

**Vector Database**
A database optimized for storing and searching vectors.

**Vector Store**
A system for storing and retrieving embedded documents.

**VertexAI**
Google Cloud's AI platform.

**Vision**
The ability of AI models to understand and analyze images.

## W

**Webhook**
A method for one system to send real-time data to another.

**Whisper**
OpenAI's speech recognition model.

**Workflow**
A sequence of steps or tasks to accomplish a goal.

## X

**xAI**
AI company founded by Elon Musk, creator of Grok.

## Z

**Zero-Shot**
Performing a task without specific training examples.

**Zero-Shot Learning**
The ability to perform tasks without prior examples.
