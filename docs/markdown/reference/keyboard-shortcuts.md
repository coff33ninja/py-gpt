## üéØ Pro Tips

### Tip #1: Tab Management
```
Organize by project:
- Keep related chats in tabs
- Name tabs clearly
- Use color coding (if available)

Keyboard: Ctrl + Tab to cycle
```

### Tip #2: Response Control
```
For faster responses:
- Use ‚Üë arrow to retry with different settings
- Reduce max_tokens temporarily
- Use faster models (e.g., 2.5-flash vs GPT-4)

For better responses:
- Increase temperature slightly
- Use extended thinking mode
- Add more context
```

### Tip #3: Copy Code Blocks
```
Quick copy:
1. Hover over code block
2. Click copy button (top-right)
3. Paste anywhere

Or: Ctrl + C after selecting
```

### Tip #4: Export Conversations
```
Save important conversations:
1. Right-click on chat tab
2. Select Export
3. Choose format (Markdown, PDF, JSON)
4. Save to folder
```

### Tip #5: Search History
```
Find old conversations:
1. Use Ctrl + F to search current chat
2. Use search panel for all chats
3. Filter by date, topic, model
```

### Tip #6: Model Switching Mid-Chat
```
Switch models without losing context:
1. Select model from dropdown
2. Continue conversation
3. AI adapts to new model

Note: Some context may be lost if new model has smaller window
```

### Tip #7: Streaming Responses
```
Real-time response display:
- Responses appear as they're generated
- Click 'Pause' to stop early
- Click 'Regenerate' to retry

Speed up display: Reduce UI update frequency in Settings
```

### Tip #8: Token Monitoring
```
Monitor token usage:
- Check bottom-right token counter
- Adjust response length if needed
- Choose cheaper models for long sessions
```

### Tip #9: Custom Shortcuts
```
Create your own shortcuts:
1. Settings ‚Üí Keyboard
2. Find action
3. Assign shortcut
4. Restart app (sometimes needed)

Tip: Use Shift for alternate actions
```

### Tip #10: Night Mode
```
Eye strain reduction:
- Enable dark mode: Ctrl + D
- Increase font size: Ctrl + +
- Reduce brightness: Settings ‚Üí Interface
- Use warm color theme
```

## üîß Customizing Shortcuts

### In Settings
1. Open Settings: `Ctrl + ,`
2. Go to `Keyboard` tab
3. Find shortcut to change
4. Click and enter new combo
5. Click `Save`

### Conflict Resolution
```
If shortcut conflicts with system:
- Check system-wide shortcuts first
- System shortcuts usually have priority
- Try Ctrl + Alt + [key] instead
- Or Ctrl + Shift + [key]
```

## üéØ Common Issues & Solutions

### Issue: Shortcuts not working
- Check if conflicting with system shortcuts
- Try alternative: e.g., `Alt + N` instead of `Ctrl + N`
- Restart application
- Check Settings ‚Üí Keyboard

### Issue: Typing too slow
- Increase typing repeat rate in system settings
- Check hardware (wireless connection issues?)
- Disable effects: Settings ‚Üí Interface ‚Üí Disable animations

### Issue: Can't select text
- Check if in input mode
- Click to focus text area first
- Try triple-click to select all

### Issue: Copy/paste not working
- Check OS clipboard is working
- Try Ctrl + Shift + C for formatted copy
- Restart app if persistent
