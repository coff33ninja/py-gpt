## ğŸ› Troubleshooting

### Microphone Not Working

**Check:**
1. Microphone connected?
2. Microphone selected in settings?
3. Permission granted?
4. Volume level sufficient?

**Fix:**
1. Test system microphone first
2. Restart PyGPT
3. Check system settings
4. Try different mic

### Voice Output Not Working

**Check:**
1. Speaker connected?
2. Speaker selected in settings?
3. Volume not muted?
4. Provider supports voice?

**Fix:**
1. Test system speaker first
2. Select speaker in settings
3. Increase volume
4. Restart app

### Transcription Errors

**Improve accuracy:**
1. Speak more clearly
2. Reduce background noise
3. Slower speech
4. Good quality microphone
5. Right language selected

### Voice Quality Poor

**Improve quality:**
1. Better microphone
2. Quieter environment
3. Closer to microphone
4. Clear pronunciation
5. Slower pace
